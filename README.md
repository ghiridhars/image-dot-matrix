# Image to Dot Matrix Converter

A simple browser-based tool that converts images into dot matrix representations.

## Usage

Open `index.html` in any modern browser. No server required.

1. Upload an image
2. Adjust settings (dot size, spacing, shape, colors)
3. Download or copy the embeddable code

## Features

- Multiple color modes (Full Color, Grayscale, Black & White, Custom)
- Adjustable dot size and spacing
- Different dot shapes (Circle, Square, Diamond)
- Brightness-based dot sizing
- Export as PNG or embeddable SVG/HTML

## Tech

Pure HTML, CSS, and JavaScript using the Canvas API.
